# PockyToASCII
A CLI Program and Library to transform raster images to ASCII text

## Usage :
```
Usage: java -jar PockyToASCII.jar
 -f,--format <arg>        Format of the output image. Values are ASCII,
                          ANSI or HTML. Default is ASCII.
 -h,--help                This utility takes an image and outputs an ASCII
                          Art version of it, in either raw ASCII, or with
                          ANSI codes, or in HTML.
 -height,--height <arg>   Desired height of the output image. Default is
                          -1
 -i,--input <arg>         Path to the input file. Default is
                          Image/pocky.jpg
 -o,--output <arg>        Path to the output file
 -v,--version             Version of the application
 -width,--width <arg>     Desired width of the output image. Default is
                          300
```
## Future

Leverage this project as a library and host it as a web service behind a RESTful API using Spring.
